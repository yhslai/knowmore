# Eldorian Language: Luminary Script and Grammar

## Language Statistics

| Property | Value |
|----------|-------|
| Official Name | **Lumari** (also called "Luminary Speech") |
| Alternate Names | Eldorian, High Tongue |
| Native Speakers | ~12.5 million |
| Second Language Speakes | ~3.2 million |
| Writing System | Three forms: Taper Script, Common Script, and Merchant Cipher |
| Language Family | **Eldorianic** (isolated language family, no known relatives) |

## Historical Development

### Early Eldoria (1000-1247)
Original Eldorian was a **syllabic language** where each character represented a syllable combination. The script consisted of geometric shapes that could be written vertically or horizontally depending on context.

### The Script Reform (1247-1300)
Following the Unification Wars, scholars developed three parallel writing systems:

#### 1. Taper Script (Formal)
- Used for official documents and legal texts
- Contains 156 base glyphs
- Written from left to right
- Each character encodes additional information about sentence function

#### 2. Common Script (Daily Use)
- Simplified for general communication
- 89 base glyphs
- Written vertically like traditional Eastern scripts
- Used in schools and everyday correspondence

#### 3. Merchant Cipher (Commercial)
- Highly abbreviated for trade records
- 45 base glyphs
- Written top-to-bottom in columns
- Contains specialized symbols for prices, weights, and quantities

## Phonology

### Vowel System

| Category | Vowels | Notes |
|----------|--------|-------|
| Front | ɛ, e, i | Similar to French é, è, i |
| Central | ə | Schwa sound |
| Back | ɒ, o, u | Deep, rounded sounds |

### Consonant System

Eldorian consonants include some rare sounds not found in most major languages:

$$ \{ /ɬ/, /ʁ/, /tɬ/, /ʔ/, /x/ \} $$

The voiceless lateral fricative **/ɬ/** appears in many common words and is pronounced similar to "lh" in Welsh or "tl" in Nahuatl.

## Grammar Structure

### Word Order
Eldorian follows a strict **SOV (Subject-Object-Verb)** structure:

$$ \text{[Subject] + [Object] + [Verb] + [Particle]} $$

Example:
- **English**: "The merchant sells goods"
- **Eldorian**: "The merchant goods sells"

### Case Marking
Nouns in Eldorian are marked for four grammatical cases:

| Case | Particle | Function |
|------|----------|----------|
| Nominative | -a | Subject |
| Accusative | -i | Direct Object |
| Genitive | -o | Possession |
| Dative | -e | Indirect Object |

### Tense and Aspect
Eldorian uses a complex **aspect system** rather than simple tenses:

```
Aspect Type    | Marker    | Usage Context
───────────────|───────────|──────────────────────
Completed      | -n        | Past actions with result
In Progress    | -r        | Ongoing actions
Habitual       | -l        | Repeated/scheduled actions
Future         | -t        | Anticipated events
Perfective     | -s        | Actions with full completion
```

## Writing System Details

### Taper Script Glyph Structure

Each Taper Script glyph has three components:

$$ G_{glyph} = H_{head} + B_{body} + V_{value} $$

- **Head** indicates the phonetic base
- **Body** indicates the grammatical function
- **Value** indicates emphasis or tone

### Number System
Eldoria uses a **base-60 (sexagesimal)** number system inherited from ancient mathematical traditions:

| Number | Glyph Example | Notes |
|--------|---------------|-------|
| 1 | ● | Single dot |
| 6 | ◎ | Ring symbol |
| 12 | ◇ | Diamond form |
| 60 | △ | Triangle indicates new position |

### Directional Writing
The writing system supports multiple orientation modes:

$$ D_{mode} \in \{ \text{vertical-top}, \text{vertical-bottom}, \text{horizontal-left}, \text{horizontal-right} \} $$

Each mode carries specific cultural weight:
- **Vertical-top**: Most formal (legal documents)
- **Vertical-bottom**: Religious/scientific texts
- **Horizontal-left**: Daily correspondence
- **Horizontal-right**: Commercial records

## Cultural Language Features

### The Silence of Respect
When speaking to elders or officials, speakers must observe a **linguistic courtesy protocol**:
1. Lower vowel volume
2. Slow syllable articulation by 15%
3. Avoid consonant clusters like /tɬ/ and /ɬ/
